Code for novel finite-volume averaged-based methods for solving the degenerate parabolic Generalized Porous Medium Equation (GPME) in the following two papers:

* [Numerical artifacts in the Generalized Porous Medium Equation: Why harmonic averaging itself is not to blame](https://www.sciencedirect.com/science/article/pii/S002199911830086X), *Danielle C. Maddix, Luiz Sampaio, Margot Gerritsen*, Journal of Computational Physics, 361, pp. 280-298, 2018.
